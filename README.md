Scripts 01–03 use SMC++ to estimate the demographic history for each population.

Scripts 04–08 use Pyrho to reconstruct population-specific recombination maps. The dataset is divided into two subsets, and three replicates are generated for each subset.

Scripts 09–10 generate concatenated recombination maps for each replicate.

 - Mean_Recombination_20kb.R merges recombination windows into 20 kb intervals.

 - Mean_Recombination_total.R merges recombination windows into 100 kb intervals.

Scripts 11–12, along with Plot_Me.R and Plot_Ms.R, calculate the PRDI and Δr metrics (as described in Talbi et al. 2025).

Scripts 13–15 identify recombination coldspots and hotspots using 40 kb windows.
